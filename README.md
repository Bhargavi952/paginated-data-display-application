# Data-display-application

    
## Run Locally

Clone the project

```bash
 https://github.com/Bhargavi952/paginated-data-display-application.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
  
## Tech Stack

-  React 
-  Redux
